# Лаборатория идей: ядро, кураторы и лабиринт

## Описание
Проект представляет собой мини‑вселенную идей с автономными блоками: ядро, кураторы и лабиринт альтернативных формулировок.  
Позволяет безопасно тестировать идеи, генерировать новые маршруты и расширять функционал без разрушения системы.

## Принципы работы
1. **Блоки автономны** — изменения внутри блока не ломают другие.  
2. **Интерфейсы стабильны** — способ вызова блока и его вход/выход не меняются, даже если внутренности обновляются.  
3. **Вес и фильтры** — кураторами присваивается значимость каждой идеи перед обработкой ядром.  
4. **Лабиринт** — генерирует альтернативные маршруты, но не изменяет исходную идею.  
5. **Экспериментальные ветки** — новые версии блоков тестируются отдельно, main остаётся стабильным.

## Структура проекта
- **core/** — Ядро (обработка идей с учётом веса)  
- **curators/** — Кураторы и фильтры  
- **labyrinth/** — Генерация альтернативных маршрутов  
- **experiment/** — Тестирование новых идей и функций  
- **main.py** — Точка запуска экспериментов
