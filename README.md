# Лаборатория идей: ядро, кураторы и лабиринт

## Описание
Этот проект представляет собой мини‑вселенную идей с самонастраивающимся ядром, системой кураторов и лабиринтом альтернативных формулировок.  
Позволяет безопасно проводить эксперименты, генерировать новые маршруты мыслей и тестировать идеи без риска нарушить основную систему.

## Структура проекта
- **core/** — Ядро платформы (управление равновесием и потоками идей)
- **curators/** — Фильтры/кураторы, проверяющие соответствие идей базовым правилам
- **labyrinth/** — Генерация альтернативных маршрутов и формулировок
- **experiment/** — Ветки для тестирования и прототипирования
- **main.py** — Точка запуска экспериментов

## Как использовать
1. Добавляем новые идеи через `main.py` или экспериментальные модули.
2. Каждая идея проходит проверку кураторов → обработку ядром → генерацию альтернативных маршрутов в лабиринте.
3. Экспериментальные изменения делаем в ветке `experiment`, а стабильные результаты сливаем в `main`.

## Пример идеи для эксперимента
Идея для эксперимента — **однострочный текст**:
"Паук кормит бабочек"

### Поток идеи
Идея → Кураторы → Ядро → Лабиринт → Новые маршруты

### Результат лабиринта
"Паук кормит бабочек — сияние нити" 
"Паук кормит бабочек — вихрь мыслей" 
"Паук кормит бабочек — кубик энергии"

## Ветки проекта
- `main` — стабильная версия ядра
- `experiment` — тестирование новых алгоритмов, формулировок и экспериментов
